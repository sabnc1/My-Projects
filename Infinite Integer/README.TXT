PROJECT TITLE: Infinite Integer
PURPOSE OF PROJECT: Implement a new data structure called infinite integer to work with numbers too large for the java virtual machine to handle natively and illustrate understanding of object oriented programming
VERSION or DATE: 1.0
HOW TO START THIS PROJECT: To compile: javac InfiniteIntegerTester.java
To Run: java InfinteIntegerTester
AUTHORS: Sanketh Kolli, Professor Kosiyatrakul
USER INSTRUCTIONS: Compile and run the program. Will show a number of calculations the program does to verify that the Infinite Integer data structure is working as intended
